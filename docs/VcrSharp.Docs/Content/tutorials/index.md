## Learning Path

Welcome to VCR# tutorials! In this series, you'll learn to create terminal recordings by building real examples. Each tutorial takes 10-20 minutes and results in a working GIF or video.

**Start here if you're new**: Work through the tutorials in order. By the end, you'll confidently record any terminal workflow.

## Available Tutorials

### For Beginners

**[Getting Started](./getting-started.md)** (5 minutes)
Install VCR# and create your first recording. You'll generate a simple "Hello World" GIF that shows text appearing in a terminal.

**[Create a Typing Demo](./typing-demo.md)** (15 minutes)
Build an animated typing demonstration showing realistic terminal interactions. Perfect for creating polished documentation and presentation materials.

### Next Steps

**[Capture Real Command Output](./exec-commands.md)** (20 minutes)
Record actual command execution and output. You'll create a recording of a real git workflow showing authentic program behavior.

## What You'll Learn

By completing these tutorials, you'll be able to:
- Create terminal recordings from simple tape files
- Control timing, appearance, and behavior of recordings
- Choose between simulated typing and real command execution
- Produce polished GIFs and videos for documentation, demos, and tutorials
